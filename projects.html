Go Transit Schedule Viewer


The Go Transit Schedule Viewer was a project I created for myself to make it easier and faster to fetch Go Transit schedules for when I want to go home and visit my family and friends. The script is written purely in Python and has three modes:
1. Custom Directions
2. Barrie -> Union Stn
3. Union Stn -> Barrie
The script uses the requests library to fetch the data from the Go Transit website and a CSV file, then parses the data to make it easier to read, ultimately displaying it in a table with times converted to 12-hour format instead of military 24-hour format.
